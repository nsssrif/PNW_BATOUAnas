Les formations académiques

Pour devenir Designer UI/UX, il est généralement recommandé de valider un niveau Bac+3 à Bac+5.

  • Écoles de Design : Diplômes spécialisés en design interactif (comme l'ECAL, les Gobelins).
  • Universités / Informatique : Masters en ingénierie du web et IHM.
  • Bootcamps : Formations intensives orientées vers la reconversion.

Le set de compétences

Hard Skills (Technique) :

  • Outils de conception : Figma, Adobe XD, Sketch.
  • Langages d'intégration : Comprendre le HTML5 et CSS3.

Soft Skills (Humain) :

  • Empathie : Se mettre à la place de l'utilisateur.
  • Rigueur : Respect strict des systèmes de grilles et chartes graphiques.
