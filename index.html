Le Design
UI / UX.

Bienvenue sur ce site informatif réalisé dans le cadre de la spécialité NSI. L'objectif de ce projet est de présenter un métier clé du numérique : le Designer d'Interface et d'Expérience Utilisateur (UI/UX).

Ce site a pour but de vous guider à travers les différentes facettes de cette profession, des compétences requises jusqu'aux perspectives de carrière, avec une approche visuelle radicale.
